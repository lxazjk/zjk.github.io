# Machine Learning Systems

> 连接算法与硬件的桥梁，探索极致的计算效率。

随着深度学习模型的参数量呈指数级增长，单纯依靠算法创新已不足以满足算力需求。MLSys 旨在解决 AI 落地过程中的系统级瓶颈，从编译器优化到分布式训练，从异构计算到显存管理。

**本板块将涵盖：**

*   **推理引擎架构**：TensorRT, ONNX Runtime, vLLM 等主流框架的源码剖析。
*   **算子优化**：手写 CUDA Kernel，Triton 编程实践，以及矩阵乘法（GEMM）的极致优化。
*   **AI 编译器**：MLIR 基础设施，TVM 自动调优，以及计算图层面的融合与重写。
*   **大模型系统**：KV Cache 管理，PagedAttention 机制，以及分布式并行的通信优化。
