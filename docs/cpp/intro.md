# Modern C++

> 零开销抽象的艺术，兼顾控制力与生产力。

C++ 是构建上述所有高性能系统（MLSys, OS, HPC）的通用语言。现代 C++ (C++11/14/17/20/23) 引入了大量新特性，使其在保持底层控制力的同时，拥有了极强的表达能力和安全性。

**本板块将涵盖：**

*   **现代特性实践**：右值引用与移动语义，智能指针，Lambda 表达式，以及 C++20 Coroutines 和 Modules。
*   **模板元编程 (TMP)**：编译期计算，SFINAE 与 Concepts，以及 Type Traits 的深度应用。
*   **内存模型**：深入理解 `std::atomic`，内存序 (Memory Order)，以及 False Sharing 问题。
*   **设计模式与惯用法**：RAII 资源管理，CRTP (奇异递归模板模式)，以及 Pimpl 惯用法在库开发中的应用。
