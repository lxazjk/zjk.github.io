# Operating Systems

> 向下扎根，理解计算机系统的基石。

操作系统是软件世界的“上帝”，掌管着一切硬件资源的分配与调度。深入理解 OS 不仅是为了造轮子，更是为了在应用层开发时拥有透视底层的“X光眼”。

**本板块将涵盖：**

*   **Linux 内核源码**：进程调度（CFS），虚拟内存管理（VMA, Page Table），以及文件系统（VFS, Ext4/XFS）。
*   **虚拟化技术**：KVM 原理，Container (Docker/K8s) 的底层实现（Cgroups, Namespace）。
*   **并发与同步**：自旋锁、互斥锁、RCU 机制的内核级实现与无锁编程。
*   **微内核实验**：跟随 rCore 或 OSDI 教程，从零构建一个简单的操作系统内核。
