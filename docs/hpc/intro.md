# High Performance Computing

> 榨干硬件的每一滴性能，追求纳秒级的延迟。

HPC 不仅仅是超级计算机的专利，它是一种追求极致性能的工程思维。在摩尔定律放缓的今天，如何通过并行计算、向量化和缓存优化来提升吞吐量，是高性能程序员的核心竞争力。

**本板块将涵盖：**

*   **并行编程模型**：MPI 多机通信，OpenMP 多线程并行，以及 SIMD (AVX-512/NEON) 指令集优化。
*   **性能分析工具**：熟练使用 Perf, VTune, Nsight Systems 定位热点与瓶颈。
*   **体系结构感知**：理解 CPU 流水线，分支预测，Cache Coherence (MESI) 以及 NUMA 架构对性能的影响。
*   **数值计算优化**：稀疏矩阵运算，FFT 算法优化，以及大规模科学计算的工程实践。
